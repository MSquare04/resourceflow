# ResourceFlow

ResourceFlow — система управления корпоративными ресурсами, их доступностью и бронированием.

## Состав проекта

- `backend/` — Go backend с REST API, бизнес-логикой и миграциями PostgreSQL.
- `frontend/` — React/Vite frontend.
- `DEMO_DATA.md` — актуальное описание demo reset и демонстрационных данных.
- `postman/` — Postman-коллекции и окружения.

## Основные возможности

- каталог ресурсов и их типов;
- правила бронирования и рабочие окна;
- обычное и множественное бронирование;
- согласование заявок по ролям;
- управление пользователями и ресурсами;
- demo reset для локальной среды.

## Быстрые команды

Из корня репозитория:

```bash
make run
make test
make build
make demo-reset
```
